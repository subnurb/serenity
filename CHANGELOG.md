# Changelog

All notable changes to the Serenity theme will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2024-12-28

### Added
- Initial release of Serenity theme
- Natural, calm color palette with sage green primary accent
- Elegant typography with Playfair Display headings and Lato body text
- Custom SCSS styling for all PrestaShop components
- Smooth, deliberate CSS transitions (0.4s ease)
- Full RTL (right-to-left) language support
- Mobile-responsive design
- PowerShell build and package scripts
- Unix-compatible zip packaging with forward slashes

### Theme Features
- **Colors**: Cream background, sage green accents, warm charcoal text
- **Typography**: Serif headings, sans-serif body for readability
- **Components**: Styled cards, buttons, forms, navigation, footer
- **Product Display**: Elegant product cards with subtle hover effects
- **Checkout**: Clean, distraction-free checkout flow

### Technical
- Built on PrestaShop Hummingbird theme foundation
- Bootstrap 5.3.3 CSS framework
- TypeScript for JavaScript components
- SCSS with organized variable system
- Webpack build pipeline
- Compatible with PrestaShop 8.1.0+
